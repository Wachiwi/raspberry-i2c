Simple method to read the measured distance of the srf02 sensor.

Parameters
fd- This is the reference to the file handle of the opened i2c-bus.
values- This is the reference to a variable owned by the caller that will contain the measured values as an array: [0] => calculated distance by adding the lowbyte to the highbyte [1] => highbyte of measurement [2] => lowbyte of measurement
